Server Dotfiles
===============
Config files for staging/production servers

Supported Apps
---------------
- vim
- bash
- tmux
- ipython
- gem

Install
-------

``` bash
git clone https://github.com/humiaozuzu/server-dotfiles.git ~/.server-dotfiles
cd ~/.server-dotfiles
sh install.sh
```

Update
------

``` bash
cd ~/.server-dotfiles
git pull
```
